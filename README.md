# Lab-3

All standard development packages required.

The application launches into a camera with a bottom navbar with the options to select a furniture, delete the last placement and clear all.

Selection of the "Select Furniture" button will bring up a scrollable panel with furniture choices for the user to choose from.

Upon choosing a piece of furniture, a holographic projection of the furniture will be shown in the world to aid in placement. Clicking anywhere on the screen will confirm the placement.
